Dependencies
------------
1) c++11 compiler.
2) Boost c++ library
3) Linux OS.


Tesing configuration
--------------------
OS: SLES 11SP4
Compiler: GCC 4.9.3
Boost c++ library: 1.60


Building instructions
---------------------
1) Modify path to default json file (defines tests to run) in main.cc
2) Modify PBS specific data for your site in pbsscript.cc
3) make
